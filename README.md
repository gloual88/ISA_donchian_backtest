# 헤지펀드 추세전략 대시보드 (ISA 추세 농사법)

크로스애셋 ETF에 **추세추종(돈치안 돌파 + 추적손절)** 을 적용한 롱온리·무레버리지
전략의 시그널 대시보드입니다. 매일 종가 기준으로 **현재 보유 종목·손절선·
신규 매수/손절 신호**를 보여줍니다.

> ⚠️ **면책 (Disclaimer)**
> 본 프로젝트는 **투자 정보 제공·교육 목적**이며, 특정 금융상품의 매수·매도를
> 권유하지 않습니다. 백테스트는 종가기준·무슬리피지·US ETF 프록시(환처리) 가정
> 이며 환헤지비용·선물롤·추적오차·괴리율은 미반영입니다. 실제 성과는 다를 수
> 있고, **과거 수익률이 미래 수익률을 보장하지 않습니다.** 모든 투자 판단과
> 손익의 책임은 투자자 본인에게 있습니다.

## 전략 개요
- **시그널**: N=189일 돈치안 신고가 돌파 시 진입
- **청산**: 초기 고정 손절(신고가 − 0.4×채널폭) → 수익 전환 후 미드라인 추적손절
- **사이징**: loss-parity(시장당 동일 리스크) + 피라미딩(최대 2유닛)
- **방향**: 롱온리(개인 공매도 불가 반영), **무레버리지**(현금계좌 실현 가능)
- **벤치마크**: KOSPI200 매수보유 (KRW 기준)

## 실행 (로컬)
```bash
pip install -r requirements.txt
streamlit run app.py
```
브라우저에서 표시되는 주소(기본 http://localhost:8501)로 접속.

## 데이터
- 가격: Yahoo Finance (yfinance) — US ETF 프록시 + USD/KRW + KOSPI200
- 무위험수익률: Fama-French / FRED (pandas-datareader)
- 별도 API 키 불필요.

## 구성
- `app.py` — Streamlit 대시보드
- `isa_signals.py` — 시그널 코어(엔진 호출 → 매수/손절/보유/손절선)
- `mulvaney_replica.py` — 추세추종 백테스트 엔진
- `mulvaney_isa_backtest.py` — KRW 패널 구성(환처리) + ISA 유니버스
- `mulvaney_*` / `gold_*` — 백테스트·그리드서치·분석 스크립트(참고)

본 전략은 Mulvaney Capital 추세추종 방법론을 역설계·복제한 합성 모델입니다.
'헤지펀드'는 방법론의 출처를 뜻하며, 실제 펀드 상품이 아닙니다.
